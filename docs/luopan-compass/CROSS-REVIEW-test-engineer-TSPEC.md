# Cross-Review: TE — TSPEC-luopan-p2-true-north-capture

**Reviewer role:** Senior Test Engineer  
**Document reviewed:** TSPEC-luopan-p2-true-north-capture.md  
**Date:** 2026-04-25  

---

## Summary

The TSPEC is structurally sound: constructor injection is used consistently across all new classes, `MagneticFieldModel` and `Clock` interfaces are properly injected with no static or singleton access, and the core persistence tests (MigrationTestHelper, encryption raw-bytes check, force-stop resilience) are all present. However, the WMM unit-test vector in §10.1 uses generic coordinates (lat=0°, lon=0°, epoch=2025.0) rather than the pinned test vector mandated by REQ §8 TE-REQ-02 (lat=40.0°N, lon=−105.0°W, epoch=2025.5, expected declination +8.93°E ±0.1°), which is a blocking correctness gap. Additionally, §10 has no test class covering AT-B (offline declination accuracy), AT-C (cached-location label), AT-D (no-GPS dialog), or AT-F (WMM interference baseline upgrade), leaving a material slice of FSPEC §7 acceptance tests without any mapped test; and a filename mismatch in the encryption test will cause it to fail at runtime. These issues are addressable in one revision cycle.

---

## Recommendation: Need Attention

The TSPEC is well-structured and demonstrates solid architectural discipline — constructor injection throughout, correct seam design for `MagneticFieldModel` and `Clock`, and all three mandated persistence tests are present. However, two blocking issues prevent approval: the WMM unit-test vector does not use the REQ-pinned coordinates, and the encryption test has a database filename that diverges from the FSPEC acceptance criterion. Beyond the blockers, AT-B, AT-C, AT-D, and AT-F have no mapped test at any level, `CompassViewModel` has no unit test, and BR-CAP-08 concurrency has no explicit test. These must be resolved before implementation begins to avoid test gaps that are expensive to backfill. A targeted revision addressing findings TE-T-01 through TE-T-05 is required; TE-T-06 through TE-T-12 are strongly recommended in the same pass.
